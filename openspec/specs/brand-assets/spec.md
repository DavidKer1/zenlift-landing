# brand-assets Spec

## Purpose

Define how Zenlift brand assets appear in browser icons and visible landing page brand areas.

## Requirements

### Requirement: Zenlift brand asset consistency

The web project SHALL use the Zenlift source mark for browser icons and visible brand areas.

#### Scenario: User views the landing page

- **WHEN** the navbar or footer is rendered
- **THEN** the Zenlift mark appears next to the Zenlift name.

#### Scenario: Browser requests app icons

- **WHEN** browser icon assets are requested
- **THEN** the assets represent Zenlift rather than placeholders.
